# Macrohired - Modern HR Solutions

A premium, modern human resource consulting and staffing web application landing page built with Next.js, React, TypeScript, Tailwind CSS, and Framer Motion. 

## Project Overview

Macrohired is designed to showcase high-end workforce placement services, organizational planning, employee engagement strategies, and leadership insights. It features interactive panels, scrolling elements, and animated cards built to create a premium user experience.

### Key Features
* **Hero Banner**: High-impact modern grid visual showing core service taglines and CTAs.
* **Services Grid**: Interactive categories illustrating recruitment, staffing, consulting, and compliance areas.
* **About Banner**: Corporate mission card with vertical features layout and profile media highlights.
* **Stats Panel**: A 2x2 statistics grid (mobile responsive) displaying company achievements.
* **Portfolio Showcase**: Filterable showcase of client case studies and project success logs.
* **Why Choose Us**: Skew-wobble animated value badges highlighting company reliability.
* **Our Approach**: Sticky sidebar walking users step-by-step through our consultation workflow.
* **Team Members**: Modern card layout displaying specialists with custom floating share triggers and a bottom tagline brand marquee.
* **Client Testimonials**: A vertical rolling review feed displaying 3 reviews on desktop and a single review carousel on mobile.
* **FAQ Accordion**: Toggleable answers to common questions with chevron indicators.
* **Smart Tracking**: Form section containing tracking numbers entry alongside a grid statistics sub-banner.
* **CEO Talks Podcast**: Horizontal sliding carousel showcase featuring episode play links.

## Technologies Used
* **Framework**: [Next.js](https://nextjs.org/) (App Router)
* **Language**: [TypeScript](https://www.typescriptlang.org/)
* **Styling**: [Tailwind CSS](https://tailwindcss.com/)
* **Animations**: [Framer Motion](https://www.framer.com/motion/)
* **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/marediasoftm18/new-macrohired.git
   cd new-macrohired
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the live site.

### Build and Deploy

To create a production build:
```bash
npm run build
```

To run the built production server locally:
```bash
npm start
```
